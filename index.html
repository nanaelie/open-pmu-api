open-pmu-api

Open Source • Gratuit

Open PMU API est une API REST open source conçue pour permettre l’accès structuré aux arrivées officielles des courses PMU. Elle offre un moyen simple, rapide et fiable d’interroger des données historiques détaillées sur les courses hippiques.

L’API permet de consulter les arrivées des courses PMU en les filtrant par date, nom d’hippodrome ou nom de prix. Chaque réponse fournit des informations complètes incluant le type de course, le montant, la distance, le nombre de partants, les non-partants, l’arrivée officielle ainsi que les détails descriptifs de l’épreuve.

Conçue pour les développeurs, analystes de données et passionnés de courses, Open PMU API facilite l’intégration des données hippiques dans des applications, outils statistiques, dashboards ou projets de recherche.

Période des données : 22/01/2004 → 13/02/2026

Dernière mise à jour : 13/02/2026

Fonctionnalités

  • Recherche par date
  • Recherche par hippodrome
  • Recherche par prix

Utilisation

GET https://open-pmu-api.vercel.app/api/arrivees
date=DAY/MONTH/YEAR
prix=PRIX
hippo=HIPPODROME

Exemples d'appel

GET https://open-pmu-api.vercel.app/api/arrivees?date=12/02/2026
GET https://open-pmu-api.vercel.app/api/arrivees?prix=Prix jules roucayrol
GET https://open-pmu-api.vercel.app/api/arrivees?hippo=Cagnes Sur Me
